### Customers

To see a list of all your customers to whom you have issued licenses, select **Customers** in your side bar menu.

![PT-Customers-Menu-Table.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/5ebef3fb25cd.jpg)

The Customers page is displayed from which you can [add a customer](/docs-test/titanhq/platform/docs/add-a-customer-account/ "Add a Customer Account") and [delete a customer account](/docs-test/titanhq/platform/docs/delete-a-customer-account/ "Delete a Customer Account"). You can also view and edit customer details by going to the **Actions** column:

-   Select the view ![view-icon-small.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/4f7d1bd09d2e.jpg) icon for a customer to drill down to the customer level to view and further manage that customer.

-   Select the edit ![edit-icon-small.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/61c7eed9145b.jpg) icon for a customer to edit that customer's details.

Use the Search ![Plat-Customers-Search.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/65980ed039b1.jpg) box to search the list of customers.

Columns that can be sorted display arrows ![sort-arrows.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/b903746dee2c.jpg) beside the column title. Select the arrows to sort a column in ascending or descending order.

Select the column ![ES-Column-Reorder-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/9ff217c6917f.jpg) icon to manage table columns. With the exception of UUID, all columns are visible and arranged in a specific order by default. You can add UUID to the display and rearrange the order by doing the following:

![Plat-Cust-Column-Reorder.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/eff90e409558.jpg)

-   Slide the toggles off and on to hide or display columns.

-   Use the reordering ![ES-Reordering-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/75ef450eda3f.jpg) icon to move columns up and down the list. This changes the left-to-right display. An item at the top of the list, for example, means that it will be the first column on the left. Select **Restore default order** to restore the columns to their default positions.

-   Resize columns by hovering your cursor over the column boundary you want to adjust until it becomes a resize ![ES-Resize-Cursor.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/3b077003c7a7.jpg) cursor. Drag the boundary until the column is the width you want.

:::note
Note that any changes you make here will remain, even if you leave the page.
:::

The following columns are available:

-   _Customer_: Name of your customer.

-   _UUID_: Customer's Universal Unique Identifier, which clearly identifies each specific customer. This column is not enabled by default and can be turned on by selecting the column ![ES-Column-Reorder-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/9ff217c6917f.jpg) icon.

-   _Description_: A short description of your customer.

-   _Active Users_: These are M365 active licensed users. The number is updated daily based on the active user license.

-   _Licenses Issued_: The total number of licenses issued by you to all of your customers.

-   _Date Created_: Month, day and year that your customer was issued licenses and added to the system.

-   _Solutions_: Icons displayed in bold show solutions that are licensed to your customers.

    -   ![Plat-PhishTitan-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/3ce43f8a1004.jpg) _Email Security_: TitanHQ's email security solution, offering protection from targeted threats that arrive in a customer's inbox.

    -   ![Plat-PhS-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/1ce238cf6ee7.jpg) _Phishing Simulation_: MSPs can set up and monitor phishing campaigns for their customers, helping them to recognize deceptive lures and phishing threats.

    -   ![Plat-SAT-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/73bef9b57fb4.jpg) _Security Awareness Training (SAT)_: MSPs can schedule and monitor security awareness training for their customers, enabling them to become more aware of information security risks.

    -   ![Plat-ArcTitan-icon.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/94ce498fe6f4.jpg) _Email Archiving_: TitanHQ's automated email archiving solution.

-   _Connection_: Indicates if a customer is connected to a Microsoft 365 account. The ![Plat-Microsoft-365-Indicator.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/9249643b6edc.jpg) label shows which customers are connected; the ![Plat-Add-Connection-Link.jpg](/docs-test/imported-assets/titanhq-platform/69428-customers/c8fb2e63e50a.jpg) link highlights those customers who are not.

    Select the **Add connection** link to begin [Connecting and Configuring a Customer Account](/docs-test/titanhq/products/phishtitan/docs/email-security/connect-and-configure-a-customer-account/ "Connect and Configure a Customer Account").

-   _Actions_: View and edit customer details.

To add a new customer, see [Add a Customer Account](/docs-test/titanhq/platform/docs/add-a-customer-account/ "Add a Customer Account").
