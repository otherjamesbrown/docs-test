### Administrators

Go to **Settings** > **Administrators** to view and manage SpamTitan administrators.

-   Use the search ![admin-search-box.jpg](/docs-test/imported-assets/spamtitan-skellig/59477-administrators/c025b9e18874.jpg) box to search the list of administrators.

-   Select the column ![column-icon.jpg](/docs-test/imported-assets/spamtitan-skellig/59477-administrators/264f4fd45b19.jpg) icon to display the table column names. Slide each on and off to view or hide a column.

-   To add an administrator, go to the Actions dropdown menu and select **Add an Administrator**. See [Add an Administrator](/docs-test/titanhq/products/spamtitan/docs/skellig-9/administrators-59477/ "Add an Administrator") for further details.

-   To delete an administrator, first select the box ![selected-check-box.jpg](/docs-test/imported-assets/spamtitan-skellig/59477-administrators/290997c07565.jpg) beside the administrator (or administrators) and then select the Actions dropdown menu. Select **Delete**.

-   You can enable or disable enforced 2FA from the Actions dropdown menu. If you're an MSP, see [Two-Factor Authentication on the MSP Level](/docs-test/titanhq/products/spamtitan/docs/skellig-9/two-factor-authentication-74398/ "Two-Factor Authentication") for more information. If you're a customer admin, go to [Two-Factor Authentication on the Customer Level](/docs-test/titanhq/products/spamtitan/docs/skellig-9/two-factor-authentication/ "Two-Factor Authentication").

The following information is available for each administrator:

-   _First Name_: Administrator's first name.

-   _Last Name_: Administrator's last name.

-   _Email_: Email address for administrator.

-   _Enforced 2FA_: Indicates whether mandatory (enforced) 2FA is enabled (On) or disabled (Off) for an admin.

**Editing Administrator Details**

Select the view ![view-icon-small.jpg](/docs-test/imported-assets/spamtitan-skellig/59477-administrators/4f7d1bd09d2e.jpg)icon in the Actions column for an admin to view and further manage that admin.

#### Add an Administrator

1.  Go to **Settings** > **Administrators** and from the Actions dropdown menu, select **Add Administrator**.

    ![ST-SK-add-admin.jpg](/docs-test/imported-assets/spamtitan-skellig/59477-administrators/ebb751c56b6a.jpg)

2.  Enter details for this administrator:

    -   _First Name:_ Enter the first name of this administrator.

    -   _Last Name:_ Enter the last name of this administrator.

    -   _Email:_ Enter an email address for this administrator.

    -   _Link Lock Admin:_ Select YES to allow this administrator to enable and manage Link Lock. See [Link Lock Admin](/docs-test/titanhq/products/spamtitan/docs/skellig-9/link-lock-admin-60363/ "Link Lock Admin").

        :::note
        Link Lock is an enhancement to SpamTitan and is available after the purchase of a SpamTitan Plus license.
        :::

    -   _Password:_ Enter a password for this administrator.

    -   _Confirm Password_: Confirm the password.

3.  Select **Add**.
