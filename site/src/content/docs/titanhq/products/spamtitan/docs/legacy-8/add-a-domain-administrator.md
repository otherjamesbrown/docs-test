#### Add a Domain Administrator

A Domain Administrator must be assigned for each domain in SpamTitan Cloud. Domain Administrators have access to the domain level allow and block lists, as well as all quarantined mail for that domain. The same administrator can be assigned to multiple domains.

To add a Domain Administrator:

1.  Go to **Anti-Spam Engine** > **Admins**.

2.  Click **Add…** to open the Add Domain Administrator window.

3.  Enter an email address and domain for the Domain Administrator you wish to add.

4.  Ensure Domain Administrator is selected from the Role drop-down menu.

5.  Click **Save**.
