#### Test Authentication Settings

Use Test Authentication in **Settings** > **Access/Authentication** to ensure that your authentication settings are correct. Enter the Email Address and Password of a user to test. SpamTitan Cloud will determine the authentication method to use for that domain and validate the supplied password.
