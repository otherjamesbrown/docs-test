#### Disable M365 Outlook Report Add-in

Once you have installed TitanHQ for Outlook, you can disable the built-in M365 reporting tool from your Outlook client. You can do this by logging into Microsoft Exchange and configuring the User Reported Settings in the Microsoft Defender portal to use a non-Microsoft add-in.

1.  Go to **Defender** > **Settings** > **Email & collaboration** > **User reported settings**. Select **Use a non-Microsoft add-in button**.

    ![ST-0365-Button1.jpg](/docs-test/imported-assets/spamtitan-skellig/78176-disable-m365-outlook-report-add-in/bc49511b2198.jpg)

2.  Select **Save**.
