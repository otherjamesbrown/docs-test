#### About the Spam Tests used by SpamTitan

SpamTitan uses a number of first-party and third-party rulesets for capturing spam. When viewing spam tests, it can sometimes be confusing as to what the test is actually doing. The links below give descriptions of the tests used in SpamTitan, including our most common third-party rulesets, such as KAM.

:::note
Not all spam tests listed are used in _every_ SpamTitan install. A subset comes out of the box and the rest must be enabled. This is because some tests are more aggressive and may cause false positives.
:::

Any rule not found in these lists may be from a third party ruleset that we don't maintain. See [Other Third Party Rulesets](/docs-test/titanhq/products/spamtitan/docs/legacy-8/other-third-party-rulesets/ "Other Third Party Rulesets").
