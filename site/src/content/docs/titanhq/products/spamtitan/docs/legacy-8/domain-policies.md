#### Domain Policies

Use **Anti-Spam Engine** > **Domain Policies** to set and manage policy settings for your domains. When a domain is added through **Settings** > **Mail Relay** it is automatically listed in Domain Policies for management.

:::note
A default domain policy is automatically created for each domain that is added to SpamTitan Cloud.
:::

Domain policy settings are inherited by all users in that domain. However, if a separate user-based policy is created in **Anti-Spam Engine** > **User Policies** then this will override the domain policy for that user.
