##### Impersonating

Impersonating a user allows you to view the SpamTitan Cloud interface with the same permissions as that user. This is a useful tool for testing and troubleshooting. To impersonate a user:

1.  Go to **Anti-Spam Engine** > **User Policies** or **Anti-Spam Engine** > **Admins**.

2.  Click on the impersonate ![ST-701-impersonate-button.jpg](/docs-test/imported-assets/spamtitan-legacy/884-impersonating/b9a0d3e4f708.jpg) icon in the Options column on the right of the user you want to impersonate.

3.  To stop impersonating and return to your own SpamTitan Cloud interface, click Stop Impersonating in the top right of the screen.
