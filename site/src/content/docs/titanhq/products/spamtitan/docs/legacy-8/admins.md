#### Admins

The Domain Group Administrator can go to **Anti-Spam Engine** > **Admins** to add, edit, impersonate or delete domain administrators.

Add a Domain Administrator

1.  Click **Add…** to open the Add Domain Administrator window.

2.  Enter an email address and domain for the Domain Administrator you wish to add.

3.  Ensure Domain Administrator is selected from the Role drop-down menu.

4.  Click **Save**.

Impersonate a Domain Administrator

Impersonating another user allows you to view the SpamTitan Cloud interface with the same permissions as that user. This is a useful tool for testing and troubleshooting. To impersonate a domain administrator:

1.  Click the impersonate ![ST-701-impersonate-button.jpg](/docs-test/imported-assets/spamtitan-legacy/885-admins/b9a0d3e4f708.jpg) icon in the Options column on the right of the user you want to impersonate.

2.  To stop impersonating and return to your own SpamTitan Cloud interface, click Stop Impersonating in the top right of the screen.

Edit a Domain Administrator

1.  Click the edit ![ST-701-edit-button.jpg](/docs-test/imported-assets/spamtitan-legacy/885-admins/7455a85c9b68.jpg) icon in the Options column on the right of the user you want to edit.

2.  Edit the domain administrator role.

3.  Click **Save**.

Delete a Domain Administrator

-   Click the delete ![ST-701-delete-button.jpg](/docs-test/imported-assets/spamtitan-legacy/885-admins/3b2dc809e314.jpg) icon in the Options column on the right of the user you want to delete.
