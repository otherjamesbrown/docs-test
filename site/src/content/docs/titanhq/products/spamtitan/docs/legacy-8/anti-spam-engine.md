### Anti-Spam Engine

SpamTitan Cloud has extensive content scanning and message filtering capabilities that allow you to apply corporate filtering policies to all messages that enter or leave your organization on a per-domain or per-recipient basis.

Go to the **Anti-Spam Engine** tab to set and manage these policies and permissions for SpamTitan Cloud.

The Domain Group Administrator can set policies at the domain and user level and will see tabs for Domain Policies, User Policies, Roles & Permissions, Admins and Domain Group.

A Domain Administrator can set policies at the user level and will see tabs for User Policies and Roles & Permissions.
