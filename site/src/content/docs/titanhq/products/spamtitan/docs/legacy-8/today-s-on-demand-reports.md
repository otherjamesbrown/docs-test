#### Today’s On-Demand Reports

Go to **Reporting** > **Todays Reports** to view any on-demand reports that were requested today in **Reporting** > **Reports**. Reports are purged from here daily. If you wish to maintain a report for longer, it can be archived.

The Report displays the date the report ran, the type of report, the period the report covers and on which node the information was collected from.

The following actions can be performed for each report:

-   _Download_: Click either Text, Excel or PDF to download that format.

-   _Archive_: Click the Archive link to make the report available in **Reporting** > **Archived Reports**.

-   _View_: Click the view icon ![ST-701-view-button.jpg](/docs-test/imported-assets/spamtitan-legacy/1197-today-s-on-demand-reports/c73fce6946d8.jpg) in the Options column to view a report. The report will display below the list of reports.

-   _Delete_: Click the delete icon ![ST-701-delete-button.jpg](/docs-test/imported-assets/spamtitan-legacy/1197-today-s-on-demand-reports/3b2dc809e314.jpg) to delete the selected report.
