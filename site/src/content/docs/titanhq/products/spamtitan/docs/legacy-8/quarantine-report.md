#### Quarantine Report

A quarantine report shows a list of emails that have not been sent to a user because they potentially contain spam or viruses. The user can decide to deliver, allow or delete the emails in the quarantine report. See [Quarantine](/docs-test/titanhq/products/spamtitan/docs/legacy-8/quarantine-overview/ "Quarantine Overview") for more information.
