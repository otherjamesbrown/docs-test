### Create a Shared Mailbox

In order to send copies of reported mails from your Outlook client to SpamTitan, you'll first need to set up a dedicated mailbox to receive the mails.

1.  Log in to [Microsoft 365 admin center](https://admin.cloud.microsoft/) and in the left navigation menu, select **Show All** > **Exchange**.

2.  In the [Exchange admin center](https://admin.exchange.microsoft.com/) tab that opens, go to the left navigation panel, and select **Recipients** > **Mailboxes**. Select **Add a shared mailbox**.

    ![ST-Skellig-Shared-Mailbox.jpg](/docs-test/imported-assets/spamtitan-skellig/77791-create-a-shared-mailbox/bf08237c3b1a.jpg)

3.  In the pop-up window, complete the following fields:

    -   _Display Name_: Enter a name for the shared mailbox.

    -   _Email address_: Enter the prefix for the email address, which is report-email-cybersentriq. Then select the domain from the _Select Domain_ dropdown menu.

    -   _Alias_: This field is optional and can be left empty.

4.  Select **Create**.

Next, [create an Exchange Connector](/docs-test/titanhq/products/spamtitan/docs/skellig-9/create-an-exchange-connector/ "Create an Exchange Connector").
