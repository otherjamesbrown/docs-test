##### Enabling Anti-Spoofing

Follow the steps below to enable Anti-Spoofing for a domain that has already been added to SpamTitan.

1.  Go to **Filter Rules** > **Anti-Spoofing Settings**.

2.  Select the box ![ST-701-check-box.jpg](/docs-test/imported-assets/spamtitan-legacy/66111-enabling-anti-spoofing/d19d75015111.jpg) to the left of the domain you want to edit and click the edit ![ST-701-edit-button.jpg](/docs-test/imported-assets/spamtitan-legacy/66111-enabling-anti-spoofing/7455a85c9b68.jpg) icon or **Edit** button. Check multiple boxes to enable multiple domains at once.

3.  Select **Enable** to enable anti-spoofing for this domain, or domains.
