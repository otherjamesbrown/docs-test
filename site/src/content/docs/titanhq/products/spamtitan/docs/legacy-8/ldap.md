#### LDAP

Lightweight Directory Access Protocol (LDAP) is an open standard protocol that is commonly used to communicate with directory servers. Common examples include Microsoft Active Directory, Lotus Domino, SunOne/iPlanet Directory Server and Novell eDirectory. To learn more, visit [https://ldap.com/learn-about-ldap/](https://ldap.com/learn-about-ldap/) ![icons8-external-link-32.png](/docs-test/imported-assets/spamtitan-legacy/677-ldap/9407e2037883.png).
