##### Domain Administrator

The Domain Administrator is typically the day-to-day user of SpamTitan Cloud. The Domain Administrator has domain-level admin privileges with access to domain level allow/block lists, domain-level reporting, and history as well as all the quarantined items for the domain to which they are assigned.
