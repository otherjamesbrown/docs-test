### Test Connectivity to your Mail Server

Once a domain has been added, follow this video or the steps below to test that your mail server accepts email from SpamTitan.

1.  Go to **Overview** > **All Domains** and select a domain to test from the list of domains.

2.  Select the **Domain Configuration** tab.

    ![ST-SK-domain-configuration-send-test-email.jpg](/docs-test/imported-assets/spamtitan-skellig/61798-test-connectivity-to-your-mail-server/4354e22a599f.jpg)

3.  Select **Send Test Email** and the Send Test Email window displays.

4.  Enter a valid email address in the Email Address field and select **Send**.

5.  Verify the email has been delivered to the specified email address.
