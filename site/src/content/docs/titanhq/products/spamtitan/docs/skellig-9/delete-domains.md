#### Delete Domains

1.  Go to **Overview** and select the checkbox(es) for domains(s) you want to delete. From the Actions dropdown menu, select **Delete**.

    ![ST-SK-Delete-Domain1.jpg](/docs-test/imported-assets/spamtitan-skellig/74497-delete-domains/1aa760d5a3e7.jpg)

2.  You'll be prompted in the pop-up window to confirm your action. Select **Delete**.

    ![ST-SK-Delete-Domain.jpg](/docs-test/imported-assets/spamtitan-skellig/74497-delete-domains/c43e7b5156ea.jpg)
