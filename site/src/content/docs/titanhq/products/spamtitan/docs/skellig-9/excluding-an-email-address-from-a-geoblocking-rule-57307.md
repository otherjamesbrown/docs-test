#### Excluding an Email Address from a Geoblocking Rule

You can exclude email from geoblocking rules by creating exemptions. This allows email through even if a geoblocking rule is blocking the country it originates from. Email can be exempted based on the sender's IP, domain or email address.

Follow the steps below to create an exemption based on sender email, which is the email address that an email originates from.

1.  Go to **Filter Rules** > **Geoblocking** and select the **Exemptions** tab.

2.  Select **Add** > **Sender Email** and the Sender Email window displays:

    ![ST-SK-geoblocking-add-email-exemption.jpg](/docs-test/imported-assets/spamtitan-skellig/57307-excluding-an-email-address-from-a-geoblocking-rule/cf5c199790bb.jpg)

3.  Complete the fields as follows:

    -   _Sender Email_: Email address you want to exempt from geoblocking rules.

    -   _Comment_: An optional comment on this email exemption.

4.  Select **Add Exemption** to save this email exemption, or **Cancel** to discard.

Once an exemption has been added, it can be edited or deleted:

-   Select the edit ![edit-icon.jpg](/docs-test/imported-assets/spamtitan-skellig/57307-excluding-an-email-address-from-a-geoblocking-rule/90a487f7499b.jpg) icon in the Actions column to edit an exemption.

-   Delete an exemption by selecting the checkbox ![selected-check-box.jpg](/docs-test/imported-assets/spamtitan-skellig/57307-excluding-an-email-address-from-a-geoblocking-rule/290997c07565.jpg) beside the exemption name and selecting **Delete**. Multiple exemptions can be deleted together by selecting multiple exemptions and selecting **Delete**.
