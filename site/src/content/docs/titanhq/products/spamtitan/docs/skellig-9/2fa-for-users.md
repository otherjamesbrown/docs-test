#### 2FA for Users

If you've [set up mandatory 2FA](/docs-test/titanhq/products/spamtitan/docs/skellig-9/enabling-mandatory-two-factor-authentication/ "Enabling Mandatory Two-Factor Authentication") for your domains, you may want to disable it for some users within the domains. Likewise, if you've [disabled 2FA](/docs-test/titanhq/products/spamtitan/docs/skellig-9/disabling-mandatory-two-factor-authentication/ "Disabling Mandatory Two-Factor Authentication") for some domains, you may want to make it mandatory for certain users within those domains.

To manage 2FA for users in the domains, follow the steps below.

1.  Go to **Settings** > **Two Factor Authentication**, where the All Domains table displays a list of your domains and their Enforced 2FA status.

    ![ST-Cust-Domain-Table.jpg](/docs-test/imported-assets/spamtitan-skellig/74501-2fa-for-users/7da292f36800.jpg)

2.  Click on the domain in the table to open the user slide-out. From here you can enable mandatory 2FA or disable it for users.

    ![ST-Cust-Domain-Users-Slideout.jpg](/docs-test/imported-assets/spamtitan-skellig/74501-2fa-for-users/87c328b15780.jpg)

3.  If Enforced 2FA is Off for a user, you can enable it by selecting the checkbox(es) for those user(s) and select **Enable**.

    ![ST-Cust-User-Enable.jpg](/docs-test/imported-assets/spamtitan-skellig/74501-2fa-for-users/5af477d083e3.jpg)

    You'll be prompted in the pop-up window to confirm your action. Select **Enable**.

    If Enforced 2FA is ON for a user, you can disable it for certain users. Select the checkbox(es) for those user(s) and select **Disable**.

    ![ST-Cust-User-Disable.jpg](/docs-test/imported-assets/spamtitan-skellig/74501-2fa-for-users/3a5b6de57224.jpg)

    You'll be prompted in the pop-up window to confirm your action. Select **Disable**.

4.  Select the **X** to close the slide-out.
