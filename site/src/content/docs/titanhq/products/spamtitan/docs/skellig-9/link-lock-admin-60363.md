##### Link Lock Admin

An administrator can be enabled as a Link Lock Admin when they are being added to the system. See [Adding an Administrator](/docs-test/titanhq/products/spamtitan/docs/skellig-9/administrators-59477/ "Add an Administrator").

![ST-SK-add-link-lock-admin.jpg](/docs-test/imported-assets/spamtitan-skellig/60363-link-lock-admin/8b8c1abeb903.jpg)

You can also follow the steps below to edit an existing administrator and enable them as a Link Lock Admin.

1.  Go to **Settings** > **Administrators**.

2.  Select the edit ![edit-icon.jpg](/docs-test/imported-assets/spamtitan-skellig/60363-link-lock-admin/90a487f7499b.jpg) icon in the Actions column for an administrator and the details page for that admin displays:

    ![ST-SK-edit-admin.jpg](/docs-test/imported-assets/spamtitan-skellig/60363-link-lock-admin/bcea4a06f418.jpg)

3.  Select **Yes** for **Link Lock Admin**.

4.  Select **Save**.
