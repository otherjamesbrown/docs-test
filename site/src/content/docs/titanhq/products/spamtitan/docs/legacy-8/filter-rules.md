### Filter Rules

Go to the **Filter Rules** tab to:

-   Specify specific sender email addresses or domain names you want to allow or block for your domain or domain group.

-   Set anti-spoofing settings for your domain or domain group.

-   Manage geoblocking settings to allow or block email based on geographic location.

The Domain Group Administrator can set filter rules for the domain group and will see tabs for Domain Group Block List, Domain Group Allow List, Anti-Spoofing Settings and Geoblocking.

A Domain Administrator can set filter rules for their domain and will see tabs for Domain Block List, Domain Allow List, Anti-Spoofing Settings and Geoblocking.
