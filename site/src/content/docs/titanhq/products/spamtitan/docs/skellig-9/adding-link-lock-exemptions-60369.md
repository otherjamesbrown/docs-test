#### Adding Link Lock Exemptions

You can create exemptions to exclude URLs from Link Lock by adding them to an allowed URLs list. This can be done at the MSP, customer and domain levels. Follow the steps below to create a Link Lock exemption.

1.  Go to **Filtering** > **Link Lock** and select the **Exemptions** tab.

2.  Select **Add** and the Add Allowed URL window displays.

    -   _Allowed URL_: The URL you want to be exempt from Link Lock.

    -   _Comments_: Add optional comments about this allowed URL.

3.  Select **Add**.
