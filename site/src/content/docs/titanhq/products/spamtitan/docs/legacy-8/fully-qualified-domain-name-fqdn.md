#### Fully Qualified Domain Name (FQDN)

The complete domain name for a specific computer (host) on the Internet. The FQDN, which includes the hostname and domain name, is converted into a physical IP address. For example, www.titanhq.com is the FQDN for the TitanHQ website where 'www' is the host and 'titan.com' is the domain name.
