##### Edit a User Policy

To edit an existing user policy:

1.  Go to **Anti-Spam Engine** > **User Policies**.

2.  Click the edit icon ![ST-701-edit-button.jpg](/docs-test/imported-assets/spamtitan-legacy/759-edit-a-user-policy/7455a85c9b68.jpg) in the Actions column to the right of the user policy you wish to edit.

3.  Using the table below as a reference, complete the **Edit User Policy** window.

4.  Click **Apply** to save any changes.
