###### Domain Allowed URLs

You can exclude URLs from Link Lock by adding them to an allowed URLs list. This can be done at the global, domain and user levels.

Follow the steps below to add a URL to your domain allowed URLs list. The URLs on this list are exempt from Link Lock and will not be blocked for any email received by your domain.

1.  Go to **Content Filtering** > **Link Lock** and scroll to the **Allowed URLs** panel.

2.  In the **Domains:** table, click **Add...** and the Add Allowed URL window displays.

    ![ST-add-domain-link-lock-allowed-URL.jpg](/docs-test/imported-assets/spamtitan-legacy/30226-domain-allowed-urls/1c3ccb8804bb.jpg)

3.  Complete the fields as follows:

    -   Domain: Enter your domain.

    -   URL: The URL you want to be exempt from Link Lock for this domain.

    -   Comment: An optional comment on this allowed URL.

4.  Click **Apply**.
