#### Delete Customers

1.  Go to **Overview** and select the checkbox(es) for customer(s) you want to delete. From the Actions dropdown menu, select **Delete**.

    ![ST-SK-Delete-Cust.jpg](/docs-test/imported-assets/spamtitan-skellig/74478-delete-customers/f3ab1e1f6a87.jpg)

2.  You'll be prompted in the pop-up window to confirm your action. Select **Delete**.

    ![ST-SK-Delete-Cust1.jpg](/docs-test/imported-assets/spamtitan-skellig/74478-delete-customers/268642730789.jpg)
