#### Who is the SpamTitan Cloud Guide for?

This guide can be used by the [Domain Group Administrator](/docs-test/titanhq/products/spamtitan/docs/legacy-8/domain-group-administrator/ "Domain Group Administrator") to configure SpamTitan Cloud by following the [SpamTitan Cloud setup](/docs-test/titanhq/products/spamtitan/docs/legacy-8/spamtitan-cloud-setup/ "SpamTitan Cloud Setup") steps.

The guide can also be used by the [Domain Administrator](/docs-test/titanhq/products/spamtitan/docs/legacy-8/domain-administrator/ "Domain Administrator") as a reference tool when using SpamTitan Cloud.
