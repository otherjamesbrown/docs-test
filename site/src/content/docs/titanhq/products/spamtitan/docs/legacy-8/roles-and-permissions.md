#### Roles and Permissions

**Anti-Spam Engine** > **Roles and Permissions** show the roles available to edit and manage for each level of log in:

-   The Domain Group Administrator sees tabs for Domain Administrator Roles and User Roles.

-   A Domain Administrators sees User Roles.

From this tab you can create and modify which tabs Domain Administrators and Users have access to when they log in.

A default Domain Administrator role is listed under the Domain Administrator Roles and a default User role is listed under User Roles. These default roles can only be viewed, they can not be edited or deleted.

[Click here](/docs-test/titanhq/products/spamtitan/docs/legacy-8/edit-add-or-delete-a-role/ "Edit, Add or Delete a Role") for instructions to edit, add or delete a custom role.
