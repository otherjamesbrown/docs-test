#### Searching Email History

Go to the **History** tab and follow the steps below to search your mail history.

:::note
The search you perform is preserved until you select **Clear Search**. This means that you can leave this page, and when you return, your search criteria and your search results are still visible.
:::

1.  Select **Advanced Filter** to expand the advanced search options.

    ![ST-SK-Advanced-Filter-Fields.jpg](/docs-test/imported-assets/spamtitan-skellig/66427-searching-email-history/194cb53c1b4d.jpg)

2.  Filter your search using the following options:

    -   _Date Range_: Select a date range for your search. Overwrite the displayed date or use the calendar ![calendar-icon.jpg](/docs-test/imported-assets/spamtitan-skellig/66427-searching-email-history/e023cf1f64dd.jpg) icon to select a date from the calendar.

    -   _Recipient (To)_: Enter a full email address or perform wildcard searches with an asterisk (\*). For example, entering _\*@example.com_ shows email sent to all users with the _example.com_ domain.

    -   _Sender (From)_: Enter a full email address or perform wildcard searches with an asterisk (\*). For example, entering _\*@example.com_ shows email sent from all users with the _example.com_ domain.

    -   _Subject_: Search based on a message subject. Perform wildcard searches with an asterisk (\*).

    -   _Envelope From_: Search based on the envelope from email address. The envelope from address is a return address in the email header that is used to return bounced mail.

    -   _SpamTitan ID_: A unique message identification number that SpamTitan assigns to each email.

    -   _Source IP address_: Search based on the connecting client IP address.

    -   _Delivery Status_: Delivery status can be Any, Sent, Deferred, Bounced, Expired, Clean, Blocked Spam, Blocked Banned, Blocked Infected or Reject. Messages rejected or quarantined by SpamTitan have no delivery status assigned to them as they have not been delivered.

    -   _Message Type_: Search based on how a message was classified by SpamTitan. The default is any type, or you can select one of the available options.

    -   _Score_: The spam score assigned to an email. Messages such as rejected messages that are not analyzed for spam do not have a spam score.

3.  Select **Apply** to search History.
