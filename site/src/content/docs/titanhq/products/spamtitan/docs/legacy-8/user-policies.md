#### User Policies

**Anti-Spam Engine** > **User Policies** lists users with a user-specific policy that may differ from their domain level policy. By default, users inherit the policy set for their domain, however, a user level policy overrides the domain level policy.

It is sometimes useful to customize a user's settings. For example, if the domain policy for example.com specifies that quarantine reports are sent every weekday, a user policy for user@example.com could override this so their quarantine report is only delivered weekly.
