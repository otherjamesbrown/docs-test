#### Viewing Archived Reports

Go to **Reporting** > **Archive** to view all scheduled or on demand reports that have been archived.

![ST-SK-archive-report-table.jpg](/docs-test/imported-assets/spamtitan-skellig/61217-viewing-archived-reports/deb2d58654f5.jpg)

-   Use the Search ![search-box.jpg](/docs-test/imported-assets/spamtitan-skellig/61217-viewing-archived-reports/84196d80bff8.jpg) box to search the list of archived reports.

-   Click the column ![column-icon.jpg](/docs-test/imported-assets/spamtitan-skellig/61217-viewing-archived-reports/264f4fd45b19.jpg) icon to display the table column names. Slide each on and off to view or hide a column.

-   To delete an archived report, select the box ![selected-check-box.jpg](/docs-test/imported-assets/spamtitan-skellig/61217-viewing-archived-reports/290997c07565.jpg) beside the report (or reports) and select **Delete**.

-   To download a report, select a button in the **Actions** column. Available formats are Text, Excel and PDF.
