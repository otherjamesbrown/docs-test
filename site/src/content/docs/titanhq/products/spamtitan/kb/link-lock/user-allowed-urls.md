You can exclude URLs from Link Lock by adding them to an allowed URLs list. This can be done at the global, domain and user levels.

Follow the steps below to add a URL to your user allowed URLs list. The URLs on this list are exempt from Link Lock and will not be blocked for any email received by this user.

1.  Go to **Content Filtering** > **Link Lock** and scroll to the **Allowed URLs** panel.

2.  In the **Users:** table, click **Add...** and the Add Allowed URL window displays.

    ![](/docs-test/imported-assets/spamtitan-kb/4000186947/96a7a13d8c81.jpg)

3.  Complete the fields as follows:

    -   User: The user to exempt from LInk Lock.

    -   URL: The URL you want to be exempt from Link Lock for this user.

    -   Comment: An optional comment on this user allowed URL.

4.  Click **Apply**.
