###### Disable Link Lock in SpamTitan (Legacy)

Follow the steps below to disable Link Lock for a domain in SpamTitan (8.00 or earlier).

1.  Log into SpamTitan.

    ![old-UI-login-very-small.jpg](/docs-test/imported-assets/phishtitan-docs/65975-disable-link-lock-in-spamtitan--legacy-/d9030b5f658c.jpg)

2.  Go to **Content Filtering** > **Link Lock** and scroll to the **Configuration** panel.

3.  In the Domains table, select the edit ![ST-701-edit-button.jpg](/docs-test/imported-assets/phishtitan-docs/65975-disable-link-lock-in-spamtitan--legacy-/7455a85c9b68.jpg) icon in the Options column for the domain for which you want to disable Link Lock. You can also search for the domain from the Domains search box. The Edit Link Lock Configuration (Domain) window displays.

    The Edit Link Lock Configuration (Domain) window displays.

    ![ST-edit-link-lock-configuration.jpg](/docs-test/imported-assets/phishtitan-docs/65975-disable-link-lock-in-spamtitan--legacy-/208cc7b7e477.jpg)

4.  Select **Disable** to disable Link Lock, and then select **Apply** to save changes.
