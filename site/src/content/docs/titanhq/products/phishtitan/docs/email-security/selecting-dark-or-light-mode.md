#### Selecting Dark or Light Mode

A logged-in user can switch their interface between light, dark, or system mode. Changes in appearance take effect for the logged-in user only. Follow the steps below to change the interface appearance.

1.  Go to the Display Mode ![Plat-Display-Mode.jpg](/docs-test/imported-assets/phishtitan-docs/68155-selecting-dark-or-light-mode/fbb652c37dcd.jpg) menu.

2.  Select from **Light**, **Dark**, or **System** display modes:

    -   Light mode shows a light background with darker text

    -   Dark mode shows a dark background with lighter text.

    -   System mode defaults to your system settings selection.
