#### Email History

Go to **Email Security** > **Email History** to view emails that have been processed by Email Security.

![ES-Email-History-No-Slideout.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/e00dacefcce2.jpg)

You can manage how the table is displayed as follows:

-   Use the search box ![SAT-Search-Modules.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/a9fac70da944.jpg) to search terms in the Recipient, Sender, and Subject columns.

-   Select the filtering options ![SAT-Filtering-Options.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/fb08a58b91b8.jpg) icon to show/hide the filter options, and then apply filters based on **Date**, **Verdict**, **Status**, and **Customer**. Select the term(s), and then select **Apply**.

-   Select **Refresh** to update the table with the latest email history. Note that selecting **Refresh** maintains your current view as it does not clear filters or search terms,

-   Select the column ![ES-Column-Reorder-icon.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/9ff217c6917f.jpg) icon to manage how table columns are displayed. By default, all columns are visible and arranged in a specific order. You can change the display by doing the following:

    -   Select or deselect the checkbox beside a column name to show or hide a column.

    -   Use the reordering ![Sherman-reorder-icon-new.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/9323022e7b0e.jpg) icon to move columns up and down the list. This changes the left-to-right display. An item at the top of the list, for example, means that it will be the first column on the left. Select **Reset Order** to restore the columns to their default positions.

-   Resize columns by hovering your cursor over the column boundary you want to adjust until it becomes a resize ![ES-Resize-Cursor.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/3b077003c7a7.jpg) cursor. Drag the boundary until the column is the width you want.

-   Select the up/down ![sort-arrows.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/b903746dee2c.jpg) arrows in the column headers to sort column content.

When you select an email in the list, an Email Summary slide-out appears that provides more information about the email. See [Managing Email History](/docs-test/titanhq/products/phishtitan/docs/email-security/managing-email-history/ "Managing Email History") for details.

![ES-Email-History-Slideout.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/de5b8e578946.jpg)

If an email is available for remediation, this can be done by selecting **Actions** > **Remediate**. See [Taking Action on an Email](/docs-test/titanhq/products/phishtitan/docs/email-security/taking-action-on-an-email/ "Taking Action on an Email") for details.

![ES-Email-History-Action-Remediate1.jpg](/docs-test/imported-assets/phishtitan-docs/78398-email-history/288a61deef7e.jpg)

Email History shows the following information for each email:

-   _Date_: Date and time the email was received.

-   _Recipient_: Recipient's email address.

-   _Sender_: The sender's email address.

-   _Subject_: Subject of the email.

-   _Verdict_: Classification of the email, for example, _Clean_ or _Phishing_. See [Managing Email History](/docs-test/titanhq/products/phishtitan/docs/email-security/managing-email-history/ "Managing Email History") for details.

-   _Status_: Indicates the status of a message, such as _Delivered_ or _Banner Applied_. See [Managing Email History](/docs-test/titanhq/products/phishtitan/docs/email-security/managing-email-history/ "Managing Email History") for details.

-   _Customer_: Name of the customer.

-   _Sender IP_: IP address of the sending server.
