#### Connect and Configure a Customer Account

Follow the steps below to connect and configure a customer account.

1.  [Connect the customer account to Microsoft 365](/docs-test/titanhq/products/phishtitan/docs/email-security/connect-to-microsoft-365-62975/ "Connect to Microsoft 365").

2.  [Add a Non-Delivery Report (NDR) email address to Microsoft 365](/docs-test/titanhq/products/phishtitan/docs/email-security/add-a-non-delivery-report-email-address-62976/ "Add a Non-Delivery Report Email Address").

3.  [Add a journaling rule to Microsoft 365](/docs-test/titanhq/products/phishtitan/docs/email-security/add-a-journaling-rule-62977/ "Add a Journaling Rule").

4.  [Deploy TitanHQ for Outlook in Microsoft 365](/docs-test/titanhq/products/phishtitan/docs/email-security/deploy-titanhq-for-outlook-in-microsoft-365-66781/ "Deploy TitanHQ for Outlook in Microsoft 365").

5.  _Does this customer use Link Lock as part of their SpamTitan service?_

    -   _Yes_: Link Lock must be disabled in SpamTitan to allow Email Security to process mail accurately.

        See [Disable Link Lock in SpamTitan](/docs-test/titanhq/products/phishtitan/docs/email-security/disable-link-lock-in-spamtitan-65973/ "Disable Link Lock in SpamTitan").

    -   _No_: Continue to next step.
