#### Email Security Customer Overview

As a Customer Admin, go to **Email Security** > **Overview** to see a summary of user activity. The default view is the last 30 days, but you can change this be selecting **Last 7 Days** in the upper right corner.

![PT-Customer-Dashboard.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/b8f3f311cfec.jpg)

-   **Emails Analyzed**

    The total number of emails processed for the selected time period.

    :::note
    This number is based on the email count and not the number of recipients who received the email. For example, if an email has 5 recipients, the email is only counted once.
    :::

    ![PT-Customer-Dashboard-Emails-Analyzed.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/0d33e24d1f72.jpg)

-   **Incidents**

    Total number of phishing emails received for the selected time period in the Incidents section. Select this box to go to **Incidents**, where you can view, resolve, and manage incidents.

    ![PT-Customer-Dashboard-Incidents.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/10cec529da87.jpg)

-   **Active Users/Licenses Purchased**

    Compare the total number of your active users to the number of licenses you purchased from your MSP. Select this box to go to **License Usage** where you can monitor your relationship with your users based on licenses and usage.

    ![PT-Customer-Dashboard-Active-Users.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/e1245d1681ff.jpg)

-   **Emails Analyzed vs Incidents**

    View a comparison of emails that have been processed compared to the number of incidents received.

    ![PT-Customer-Dashboard-Emails-Anal-vs-Incidents.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/73b355d9a525.jpg)

    -   Hover your mouse over the graph to display a summary for a particular date.

        ![PT-Overview-EmailsAnal-vs-Incidents.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/7e9a44f3ba9c.jpg)

    -   Select and de-select the labels to add and remove items from the graph.

        ![PT-Overview-Labels.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/a49df7c7314d.jpg)

-   **Most Targeted Users**

    View a list of the most attacked email addresses, with the most targeted user listed first. The **\# Targeted** column indicates the number of malicious emails sent to the user.

    ![PT-Most-Targeted-Users.jpg](/docs-test/imported-assets/phishtitan-docs/67843-email-security-customer-overview/ac3d040a586c.jpg)
