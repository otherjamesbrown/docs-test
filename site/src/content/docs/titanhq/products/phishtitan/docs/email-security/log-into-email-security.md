#### Log into Email Security

Log into Email Security at [platform.titanhq.com](https://platform.titanhq.com/auth/sign-in) using the login credentials provided when your account was created.

:::note
You can configure SAML SSO to use your existing Identity Provider as an authentication method for the platform. Go to Configuring SAML SSO for details. Once you have configured SAML SSO, you can [log in using SAML SSO](/docs-test/titanhq/products/phishtitan/docs/email-security/log-in-using-saml-sso/ "Log in Using SAML SSO").
:::

When you log in, you will see your **[Dashboard](/docs-test/titanhq/platform/docs/msp-dashboard-70565/ "MSP Dashboard")**.

-   If you are a new Email Security MSP admin, continue with the remaining setup steps to add customers and complete your setup.

-   To see information about the logged-in account and to log out, select the profile ![profile-icon.jpg](/docs-test/imported-assets/phishtitan-docs/62538-log-into-email-security/5225ccde9cb8.jpg) icon.
