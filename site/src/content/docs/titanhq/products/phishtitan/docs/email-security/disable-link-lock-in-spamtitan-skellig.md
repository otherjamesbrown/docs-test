##### Disable Link Lock in SpamTitan Skellig

An MSP Admin can follow the steps below to disable Link Lock for a customer in SpamTitan Skellig (9.00 or earlier).

1.  Log into SpamTitan Skellig.

    ![skellig-login-very-small.jpg](/docs-test/imported-assets/phishtitan-docs/66814-disable-link-lock-in-spamtitan-skellig/9315fcbe3c8f.jpg)

2.  Select a customer from the All Customers table or the side bar menu to drill down to that customer level.

3.  Go to **Filtering** > **Link Lock**.

4.  Select **Disabled** to disable Link Lock for the selected customer.
