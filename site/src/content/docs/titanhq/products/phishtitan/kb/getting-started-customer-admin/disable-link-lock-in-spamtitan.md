The rewriting of URLs with Link Lock is built into the Email Security solution. Therefore, if Link Lock is enabled as part of your SpamTitan service, it must be disabled to allow Email Security to process mail accurately.

Based on your SpamTitan log on screen, select an image for steps to disable Link Lock.
