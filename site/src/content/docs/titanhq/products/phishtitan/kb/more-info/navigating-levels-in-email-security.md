The Email Security solution has two levels of access - MSP and Customer, one of which your login account is based on. An MSP admin can drill down from the MSP level to the customer level to view and manage their customer accounts. See [Levels of Access for Email Security](/docs-test/titanhq/products/phishtitan/kb/more-info/levels-of-access-for-email-security/ "Levels of Access for Email Security").

There are a number of options to move between the MSP and Customer levels:

1.  **Breadcrumb**: You can travel up and down levels using the breadcrumb at the top of the page.

    ![PT-breadcrumb.jpg](/docs-test/imported-assets/phishtitan-kb/4000199192/79cb88a603e9.jpg)

2.  **Menus**: Select or search for a customer from the dropdown menu on the left. Use the breadcrumb to navigate back to MSP level.

    ![PT-msp-menu.jpg](/docs-test/imported-assets/phishtitan-kb/4000199192/0b373d3abf33.jpg)

3.  **Overview** page: An MSP Admin can select the view ![view-icon-small.jpg](/docs-test/imported-assets/phishtitan-kb/4000199192/4f7d1bd09d2e.jpg) icon for a customer in the **All Customers** table. Use the breadcrumb to navigate back to MSP level.

    ![PT-all-customers-table.jpg](/docs-test/imported-assets/phishtitan-kb/4000199192/c633ff4ccff8.jpg)
