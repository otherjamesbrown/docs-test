Email Security scans emails and file attachments for potential threats and malicious content, which are called [Incidents](/docs-test/titanhq/products/phishtitan/kb/incidents/incidents/ "Incidents"). The emails are classified into categories associated with phishing, which are described below.

<table width="100%"><thead><tr><th><p>Email Classification</p></th><th><p>Description</p></th></tr></thead><tbody><tr><td><p>Spam</p></td><td><p>Spam refers to unsolicited emails that are sent to a large number of recipients, usually as advertising.</p></td></tr><tr><td><p>Phishing</p></td><td><p>Phishing emails attempt to trick&nbsp;people into revealing personal or confidential information which can then be used illicitly; for example, to steal a recipient's money or identity.</p></td></tr><tr><td><p>Malware</p></td><td><p>Malware, short&nbsp;for "malicious&nbsp;software", can be a file or a link in an email that is intentionally harmful to a computer, network or server. If a user interacts with the link or file, malware can infect networks and devices and steal, encrypt and delete sensitive data.</p></td></tr><tr><td><p>Marketing</p></td><td><p>Marketing emails encourage recipients to&nbsp;buy&nbsp;a&nbsp;product&nbsp;or&nbsp;service.</p></td></tr><tr><td><p>Extortion</p></td><td><p>Extortion emails seek to obtain something such as money, by using forceful or threatening language.</p></td></tr><tr><td><p>Malicious URLs</p></td><td><p>A malicious URL is a link embedded in an email that was created with the purpose of promoting scams, attacks, and frauds. When selected, malicious URLs can download ransomware, or lead to phishing or spearphishing emails.</p></td></tr></tbody></table>

The emails in these categories are flagged by Email Security and can be viewed in the Resolved tab on the [Incidents](/docs-test/titanhq/products/phishtitan/kb/incidents/incidents/ "Incidents") page. Emails reported by users using the TitanHQ for Outlook add-in, appear in the Action Needed tab and require admin attention.
